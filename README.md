## Задание 2-1. Прямая.

Создайте сущность Линия, расположенную на двумерной плоскости, которая описывается:
 Координата начала: Точка
 Координата конца: Точка
 Может возвращать текстовое представление вида “Линия от {X1;Y1} до {X2;Y2}”
Для указания координат нужно использовать сущность Точка, разработанную в задании 1.1. Создайте
линии со следующими характеристиками:
1. Линия 1 с началом в т. {1;3} и концом в т.{23;8}.
2. Линия 2, горизонтальная, на высоте 10, от точки 5 до точки 25.
3. Линия 3, которая начинается всегда там же, где начинается линия 1, и заканчивается всегда там
же, где заканчивается линия 2. Таким образом, если положение первой или второй линии
меняется, то меняется и третья линия.
4. После создания всех трех объектов измените координаты первой и второй линий, причем
сделайте это таким образом, чтобы положение третьей линии соответствовало требованиям
пункта 3.
5. Измените координаты первой линии так, чтобы при этом не изменились, координаты третьей
линии.

## Алгоритм решения:

В файле Line.cpp создаем класс Line. В приватном поле описываем атрибуты класса Point begin и Point end - координаты линии (используя ранее написанный класс Point). В публичном поле описываем методы: void set_line (int x1, int y1, int x2, int y2), с помощью котого задается начало и конец линии, string print_line(), который возвращает строку вида “Линия от {X1;Y1} до {X2;Y2}”. Также создадим методы get_begin, get_end, set_begin, set_end для изменения коорднат линии. В файле z2.cpp создадим метод Line change(Line line1, Line line2), который возращает третью линию с началом от первой линии и концом от второй.

Line from {5;10} to {25;10}<br>
Line from {1;3} to {25;10}<br>

После изменения 1 и 2 линии:<br>

## Задание 3-1. Студент.

Создайте сущность Студент, которая описывается: <br>
 Имя: строка<br>
 Оценки: массив целых чисел.<br>
 Может возвращать текстовое представление вида “Имя: [оценка1, оценка2,…,оценкаN]”<br>
Необходимо выполнить следующие задачи:<br>
1. Создать студента Васю с оценками: 3,4,5.<br>
2. Создать студента Петю и скопировать оценки Васи, присвоив содержимое поля с
оценками Васи полю с оценками Пети.<br>
3. Заменить первую оценку Пети на число 5. Вывести на экран строковое представление
Васи и Пети. Объяснить результат<br>
4. Создать студента Андрея и скопировать ему оценки Васи так, чтобы изменение оценок
Васи не влияло на Андрея.

## Алгоритм решения:

Создаем класс студент с атрибутами string name - имя студента и std::vector<int> points - оценки студента. Для хранения оценок будем использовать вектор, потому что заранее мы не знаем сколько будет оценок. Также создадим метод print, который возвращает строку вида “Имя: [оценка1, оценка2,…,оценкаN]” и метод change(int index, int point), в котором index - индекс оценки для изменения, а point - новое значение оценки.
